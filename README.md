# Netflix Content Intelligence Platform

Professional analytics dashboard with 6 intelligence dashboards, 33+ charts, and 3 ML models.

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

Opens at: `http://localhost:8501`

## What's Inside

- **6 Dashboards**: Content Universe, Performance Deep Dive, Market Opportunities, Recommendation Copilot, Predictive Forecasting, Head-to-Head Comparison
- **33+ Charts**: All interactive with insights and analysis
- **3 ML Models**: Gradient Boosting, Ridge Regression, Z-Score Anomaly Detection
- **Real Data**: 800+ titles from TMDB API

## Requirements

- Python 3.8+
- Internet connection (for TMDB API)

That's it! Just run and analyze.
